# 📘 NumPy 기초 정리

## 1. NumPy란?

NumPy는 Numerical Python의 줄임말입니다.  
**수치 계산을 빠르게 처리할 수 있도록 도와주는 파이썬 라이브러리**입니다.

### ✔ NumPy의 특징
- 대규모 숫자 데이터를 빠르게 처리
- 리스트보다 빠르고 효율적인 연산
- 수학, 통계, 과학, 인공지능에서 자주 사용

---

## 2. NumPy는 언제 사용하나요?

| 사용 상황 | 설명 |
|-----------|------|
| 수학 연산 | 벡터, 행렬 계산, 제곱, 평균 등 |
| 통계 분석 | 평균, 분산, 표준편차 계산 |
| 데이터 분석 | 대규모 데이터 처리 |
| 머신러닝 | 데이터 전처리, 수치 기반 연산 |
| 이미지 처리 | 이미지 = 숫자 배열로 구성됨 |
